控制台使用說明

歡迎使用 SGF 企劃進度控制台！以下是網頁操作功能說明：

  • 任務編輯：點擊列表最右側的 🖊️ 編輯 按鈕，可拉出右側編輯抽屜。
  • 名稱與 ID 寫回：您可以直接在抽屜頂部修改「專案項目名稱」，或修改下方「ID 欄位」，系統會自動在保存時寫回 Google 試算表對應位置。
  • 超連結公式防護：修改專案項目名稱時，Google 試算表 C 欄上的 HYPERLINK 超連結公式將會被完整保留，不被覆蓋。
  • 變更安全防手滑：若您在抽屜內修改過任何內容，點擊抽屜外部或叉叉離開時，系統會自動彈出對話框詢問是否放棄。若要直接離開，可點擊右下角「放棄修改」按鈕。
  • 進度狀態唯讀：類型、完成度、進度百分比是由 Google 試算表公式自動計算產生的，網頁上為唯讀展示。

Google Sheet API 取得與部署步驟：

  1. 開啟專案 Google Sheet 試算表 ➡️ 點選選單的「擴充功能」 ➡️ 選擇「Apps Script」。
  2. 清空原有程式碼，將專案的 google-apps-script.js 全部貼入。
  3. 點選右上角「部署」 ➡️ 選擇「新增部署」。
  4. 點擊齒輪圖示，設定類型為「網頁應用程式」。
  5. 設定「執行身分」為「我 (您的帳號)」，且「誰有權限存取」設為「任何人 (Anyone)」 ── 這一步非常重要，否則網頁無法存取 API。
  6. 點擊「部署」，並授權權限後，複製畫面上產生的「網頁應用程式網址 (Web app URL)」。
  7. 回到此網頁點點擊右上角 設定，將該 URL 貼上並儲存即可！密鑰金鑰預設為 SGF_SECURE_TOKEN_2026。
小提示： 點擊右上角的 按鈕，即可隨時手動重新載入 Google Sheet 最新的連動公式計算數據。
